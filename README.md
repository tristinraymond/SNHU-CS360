# Mobile App Portfolio Reflection

## Project Summary

The mobile application I developed was an Event-Tracking App designed to help users organize and monitor their upcoming events by storing important details such as event names, dates, times, and descriptions. The app also included a secure login and account creation system, allowing users to create personal accounts and manage their event information.

The primary goal of the app was to address the need for a simple and accessible way for users to keep track of important upcoming events in one centralized location. By creating an organized and easy-to-use platform, the app helps users stay informed, prepared, and on schedule.


## User Needs and UI Design

Several screens and features were necessary to meet user needs, including:

- **Login / Create Account Screen**  
  Allowed new users to register and returning users to securely access their accounts.

- **Event Dashboard Screen**  
  Displayed upcoming events in a clear, organized format for easy viewing.

- **Add / Edit Event Screen**  
  Gave users the ability to create, update, and manage event details.

- **Notification Permission Screen**  
  Supported SMS reminders and notifications to help users stay updated on upcoming events.

My UI design focused on:

- **Simplicity** – Clear layouts with minimal clutter  
- **Accessibility** – Easy-to-read text and straightforward navigation  
- **Efficiency** – Users could complete tasks quickly with minimal steps  
- **Consistency** – Buttons, menus, and screens followed a uniform design pattern  

These design choices were successful because they prioritized the user experience while keeping functionality practical and intuitive.


## Development Process

My approach to coding the app was to break development into smaller, manageable components and build functionality one feature at a time.

### Techniques and Strategies Used:
- Built the overall UI layout first
- Developed navigation flow between screens
- Implemented backend functionality (authentication + database storage)
- Used modular coding practices to keep code organized
- Followed an iterative development process by testing features as they were built

This strategy helped reduce bugs, improve organization, and create a strong foundation for future app development projects.


## Testing and Debugging

To ensure the application functioned correctly, I regularly tested the app using the Android emulator and verified each major feature individually, including:

- Account creation
-  Login functionality  
- Database storage  
- Event creation  
- Event editing  
- Event deletion  
- Notification permissions  

I also tested edge cases such as:

- Invalid login credentials  
- Empty input fields  
- Incorrect event information  
- Navigation errors between screens  

Testing was important because it revealed issues with navigation flow and data handling early, allowing fixes before the final iteration of the app. This process improved reliability, usability, and overall app quality.


## Innovation and Problem Solving

One of the biggest challenges during development was connecting the user interface design with backend functionality in a way that felt seamless to the user.

To overcome this challenge, I had to innovate by:

- Structuring data flow efficiently between screens  
- Designing database interactions that were reliable and easy to manage  
- Balancing technical implementation with user-centered design principles  
- Simplifying navigation so functionality felt natural and intuitive  

This challenge pushed me to think beyond design alone and focus on how the complete user experience would function from start to finish.


## Key Components

The strongest component of my application was the **database integration and event management system**.

This included:

- Secure user login functionality  
- Storing user-specific event information  
- Creating new events  
- Updating event details  
- Deleting events when no longer needed  
- Managing data efficiently through database connections  

This component demonstrated my skills in:

- Front-end development  
- Back-end logic  
- Database management  
- UI/UX design  
- Problem solving  
- Mobile development best practices
